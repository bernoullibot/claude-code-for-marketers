# CC4M Planning Documents

This directory contains strategic planning documents for Claude Code for Marketers.

## Files

- `90-DAY-PLAN.md` — Roadmap and strategic initiatives
- `CONTENT-BACKLOG.md` — Upcoming content ideas and pipeline
- `CONTENT-STATUS.md` — Current content status (idea → briefed → in-progress → review → published → promoted)
- `KIT-INTEGRATION.md` — Subscriber metrics and email performance tracking

## Integration Notes

- Kit API key needed for subscriber stats
- Content status updates should happen during heartbeat checks
- Planning docs should be updated as strategy evolves
